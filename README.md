# SADD-diabetes-app-prediction
SADD: Herramienta web de apoyo al diagnóstico de diabetes basada en Machine Learning. Predice riesgo, calcula Odds Ratios y genera recomendaciones clínicas personalizadas.

1. Descarga la carpeta del proyecto
2. Abre la carpeta con Visual Studio Code
3. pip install streamlit joblib pandas numpy scikit-learn
4. pip install streamlit pandas numpy joblib plotly
5. Ejecuta el proyecto
6. streamlit run app4.py
7. Para Cortar la ejecución es CTRL+C

También puede usar la app de forma web con el siguiente link público gracias a Cloud StreamLit:
